How to use BHelper

BHelper helps you rewrite selected text using AI.

Shortcuts:

  • Transform with Pre-prompt: Command + Control + Option + U

  • Use Selection as Prompt: Command + Control + Option + I

Steps:

  1. Select the text you want to transform.
  2. Press the desired shortcut.
  3. The transformed text will be pasted in place or copied to your clipboard.

API Keys and Credentials:

BHelper supports different AI providers. To use them, you might need to enter API keys or credentials:

  1. Click the BHelper icon in your menu bar (top right).
  2. Select "Settings."
  3. Enter your API key or credentials for the AI provider you want to use.
  4. Click "Save."
